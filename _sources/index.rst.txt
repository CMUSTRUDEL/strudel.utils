
Reference
===========

.. toctree::
   :maxdepth: 2


This package provides interface to the objects:

- `decorators` - a set of common decorators
- `email_utils` - email addresses processing functions
- `mapreduce` - utils for parallel processing
- `sysutils` - functions to run system commands
    and work with files/directories
- `versions` - utilities to work with version strings
